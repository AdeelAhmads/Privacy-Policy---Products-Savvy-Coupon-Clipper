Privacy Policy

Effective Date: July 3, 2025

The Savvy Coupon Clipper Chrome Extension is designed to help you automatically clip coupons across supported online retail and grocery websites. We are committed to protecting your privacy and want to be transparent about how this extension operates.

Information We Collect

This extension does not collect, store, or transmit any personally identifiable information (PII). All actions occur locally in your browser, and no user credentials or sensitive data are accessed or transmitted to any external server.

What the Extension Does

  • Scans and clips available digital coupons on supported grocery store websites.
  • Fetches store metadata (like API keys) from a public Google Sheet to enable proper coupon clipping.
  • Stores your extension preferences (e.g., auto-run) using Chrome's local storage API.

Permissions Explanation

  • Storage: To save your auto-run preferences and cache fetched metadata.
  • Alarms: To periodically update store metadata from a public source.
  • Host Access: Only for interacting with coupon buttons on trusted retailer websites listed in the manifest.

Third-Party Services

The extension fetches configuration data from a publicly available Google Sheets API proxy. No user data is sent during this process.

Data Security

Since no personal data is collected, there is no risk of data leaks or misuse. All logic runs entirely on your device.

Changes to This Policy

This privacy policy may be updated from time to time. All changes will be posted on this page with an updated effective date.
